## 📖 关于项目

`text-utils-go` 是一个简洁、高效的 Go 语言文本处理工具库。提供字符串转换、格式验证等常用功能，开箱即用。

**核心功能**：
- 字符串转换（驼峰与下划线互转）
- 格式验证（邮箱、手机号）
